= Dependent Libraries

Third-party packages used by the app. Updated when a dependency is added or its role changes materially.

[cols="1,1,2", options="header"]
|===
| Package | Source | Purpose

| https://github.com/nicoll-douglas/getalby-sdk[`@getalby/sdk`]
| Alby
| NWC (Nostr Wallet Connect) client — connects to Lightning wallets via Nostr relays

| https://github.com/nbd-wtf/nostr-tools[`nostr-tools`]
| nbd-wtf
| Nostr protocol utilities — event signing, relay connections, NIP-19 encoding

| https://github.com/nicoll-douglas/light-bolt11-decoder[`light-bolt11-decoder`]
| nicoll-douglas
| Lightweight BOLT-11 Lightning invoice decoder

| https://github.com/BenGWeeks/bdk-rn[`bdk-rn`]
| BDK / Peach2Peach (fork)
| Bitcoin Development Kit for React Native — HD wallet, Electrum sync, balance, transactions, UTXO management. Native Rust via FFI. Forked from https://github.com/Peach2Peach/bdk-rn[Peach2Peach] with RN 0.83 fix.

| https://github.com/bitcoinjs/bitcoinjs-lib[`bitcoinjs-lib`]
| bitcoinjs
| Bitcoin transaction construction — used for Boltz claim transactions (Taproot script-path spend)

| https://github.com/bitcoinjs/bip32[`bip32`]
| bitcoinjs
| BIP-32 HD wallet key derivation — used for zpub/ypub to xpub conversion

| https://github.com/bitcoinerlab/secp256k1[`@bitcoinerlab/secp256k1`]
| bitcoinerlab
| Pure JS secp256k1 elliptic curve (replaces WASM-based `tiny-secp256k1` for React Native)

| https://github.com/paulmillr/noble-curves[`@noble/curves`]
| paulmillr
| Audited elliptic curve cryptography — secp256k1 Schnorr signatures (BIP-340)

| https://github.com/paulmillr/scure-btc-signer[`@scure/btc-signer`]
| paulmillr / scure
| Audited Bitcoin signer utilities — MuSig2 (BIP-327) key aggregation for Boltz Taproot internal keys. MIT-licensed; Boltz's own `boltz-core` uses the same library under the hood but is AGPL-3.0, so we depend on this directly.

| https://github.com/a-konst/react-native-toast-message[`react-native-toast-message`]
| a-konst
| Toast notifications — used to surface swap recovery progress/failure at the top of the screen

| https://github.com/paulmillr/noble-hashes[`@noble/hashes`]
| paulmillr
| Audited cryptographic hash functions — SHA-256, RIPEMD-160

| https://github.com/nicoll-douglas/bs58check[`bs58check`]
| nicoll-douglas
| Base58Check encoding/decoding — used for xpub/ypub/zpub version byte conversion

| https://github.com/feross/buffer[`buffer`]
| feross
| Node.js Buffer polyfill for React Native

| https://github.com/nicoll-douglas/gorhom-bottom-sheet[`@gorhom/bottom-sheet`]
| gorhom
| Bottom sheet modal component for React Native

| https://github.com/nicoll-douglas/react-native-qrcode-svg[`react-native-qrcode-svg`]
| awesomejerry
| QR code generation as SVG for React Native

| https://github.com/nicoll-douglas/react-native-svg[`react-native-svg`]
| react-native-svg
| SVG rendering for React Native — used for custom Lucide-style icons
|===
